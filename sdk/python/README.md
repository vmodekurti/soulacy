# Soulacy Python SDK

> ⚠️ **Experimental.** This SDK is not yet published to PyPI and currently has
> **no test coverage**. APIs may change without notice. Install from source and
> pin to a commit if you depend on it. A versioned PyPI release will follow once
> the test suite lands and is wired into CI.

Python SDK for the [Soulacy](https://github.com/vmodekurti/soulacy) agentic
framework — define and run agents in Python.

## Install (from source)

```bash
pip install ./sdk/python
```
